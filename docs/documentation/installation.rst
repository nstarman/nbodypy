With `pip`
**********

.. container::

	|PyPI| |PyPI Format|


The easiest way to get *nbodypy* is to install with `pip <https://pypi.org/project/nbodypy/>`_. To install with pip::

    pip install nbodypy

See the `installation instructions <https://readthedocs.org/projects/nbodypy/>`_ in the `documentation <https://readthedocs.org/projects/nbodypy/>`_ for more information.
